# 🎓 系统功能点规范化审计报告

## 【模块名称：多模态资产获取与预处理模块】

网页功能名称：视频多维度数据采集空间
对应代码逻辑：views.launch_spider_api
操作角色：创作者
业务定义：该功能属于流程的 起点
功能描述：创作者输入采集指令，启动基于 DrissionPage 的底层浏览器实例进行长连接的抖音视频特征采集作业。

网页功能名称：增量文本评论采集组件
对应代码逻辑：views.launch_comment_only_api
操作角色：创作者
业务定义：该功能属于流程的 起点
功能描述：针对指定的单点视频对象分配 DrissionPage 爬取资源，深入补漏攫取视频下方全部社交互动和情感表达数据。

网页功能名称：多模态数据精炼与系统入库
对应代码逻辑：views.run_clean_data_api
操作角色：系统后台
业务定义：该功能属于流程的 起点
功能描述：交由 UnifiedPersistenceManager 捕获已采集的 CSV，进行 Django ORM 隔离入库和深度自然语言洗噪并关联原始视频模型对象。

## 【模块名称：资产调度与数据池管理模块】

网页功能名称：多模态资产精炼集市 / 数据仓库
对应代码逻辑：views.data_warehouse
操作角色：创作者
业务定义：该功能属于流程的 处理中
功能描述：将全局或各主题辖下的原始爬虫件及精炼数据库实体文件清单直观回显在网页大盘，供创作者下发流转指令。

网页功能名称：视频特征审计视图
对应代码逻辑：views.video_list
操作角色：创作者
业务定义：该功能属于流程的 处理中
功能描述：将 Django 分页处理后的单主题视频色彩饱和度、时长和网络流行互动度做列表化渲染复查展示。

网页功能名称：评论语义核验列表
对应代码逻辑：views.comment_list
操作角色：创作者
业务定义：该功能属于流程的 处理中
功能描述：提供界面分页表格审查每一条文本评论经由 TF-IDF 及 SnowNLP 分析得出的精准情感倾向打分（极性数值）。

网页功能名称：课题数据物理级擦除
对应代码逻辑：views.delete_theme
操作角色：创作者
业务定义：该功能属于流程的 终点
功能描述：向 Django ORM 发出指令，级联地物理消灭某个工作主题下附属的庞大 Video 与 Comment 数据群集。

网页功能名称：课题资产环境隔离切换
对应代码逻辑：views.switch_theme
操作角色：创作者
业务定义：该功能属于流程的 处理中
功能描述：更改请求上下文中 Session 所记录的活跃主题标识（Theme），实现工作流的跨域无损切换与沙盒阻断环境。

## 【模块名称：智能分析与多源可视化图谱模块】

网页功能名称：受众画像特征透视图
对应代码逻辑：views.chart_user
操作角色：创作者
业务定义：该功能属于流程的 终点
功能描述：利用前端渲染框架从 Django 数据库中分组拉取用户的 IP 及互动率，构筑动态网络用户区域化偏向分布图板。

网页功能名称：内容特征量化分布矩阵
对应代码逻辑：views.chart_content
操作角色：创作者
业务定义：该功能属于流程的 终点
功能描述：绘制 BPM、亮度等抽取的多模态特征因子对点赞数的二维热度与三维空间分布，实现核心指标关联性寻优。

网页功能名称：舆情偏好极性监控罗盘
对应代码逻辑：views.chart_sentiment
操作角色：创作者
业务定义：该功能属于流程的 终点
功能描述：聚合海量评论的情感基带阈值，呈现该主题宏观上受众态度的全景仪表盘和时序分布曲线。

网页功能名称：历史脏数据强制修复重载
对应代码逻辑：views.recalculate_sentiment_api
操作角色：系统后台
业务定义：该功能属于流程的 处理中
功能描述：后台强行迭代数据库进行全量遍历，对游离的未分析或受损语料动用最新 AI 清洗规则和过滤引擎重新处理计算极性。

## 【模块名称：AI 数据推演与前馈实验模块】

网页功能名称：AI 爆款推演沙盒控制台
对应代码逻辑：views.predict_page (UI) / views.predict_api (Core)
操作角色：创作者
业务定义：该功能属于流程的 终点
功能描述：创作者界定新的视频特征参数自变量，系统调用后台缓存中部署的机器学习预言机执行爆款潜力分数推断及特征归因。

网页功能名称：决策层模型基准演进驱动器
对应代码逻辑：views.retrain_model_api
操作角色：系统后台
业务定义：该功能属于流程的 终点
功能描述：由系统触发随机森林底层等监督学习训练集网格，抽取最新入库数据闭环更迭全局分类预测的智能判定权重。
